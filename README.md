# MC361-C_G6_Flexion_asimetrica_PC4

## Requisitos del Sistema e Instalación

Para ejecutar el programa, es necesario tener instalado **Python 3.x** y las siguientes librerías externas. Pueden instalarlas todas juntas ejecutando el siguiente comando en su terminal o símbolo del sistema (cmd):

```bash
pip install numpy matplotlib
```

Detalle de las librerías utilizadas:
**numpy: Utilizada para todo el motor matemático pesado del backend (manejo de matrices de coordenadas de los vértices, cálculo de funciones trigonométricas para la rotación de ejes y operaciones vectoriales de esfuerzos).
**matplotlib: La librería encargada de generar los gráficos de alta calidad (el degradado de colores de los esfuerzos con tripcolor/scatter y la representación geométrica del Círculo de Mohr).

Librerías nativas (No requieren instalación, pero por si acaso):
El programa también utiliza módulos que ya vienen incluidos por defecto con Python, por lo que no necesitan hacerle pip install:

**tkinter: Utilizada para construir toda la interfaz gráfica de usuario (ventanas, botones, campos de texto y contenedores con scroll).
**math: Utilizada para operaciones geométricas complementarias.
